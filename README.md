# Data Science Teaching Portfolio – Estevan Gómez

This repository contains teaching-oriented data science notebooks and materials used in university-level courses and student mentoring.

I am a Data Scientist and University Professor with over 20 years of experience teaching Data Analysis, Machine Learning, Python, and Database Systems. These notebooks reflect my practical teaching approach using real datasets, Jupyter Notebook, and applied data science methodologies.

## Repository Contents
- Data ingestion and processing using Apache Spark
- Exploratory Data Analysis (EDA)
- Data visualization with Python
- Teaching-oriented notebooks for student learning and mentoring

## Teaching & Mentorship Focus
- Explaining complex data science concepts in a clear and practical way
- Guiding students through real-world projects
- Code review and structured feedback
- Live coding sessions and technical mentoring
- Hands-on learning with Python and Jupyter Notebook

## Tools & Technologies
Python, Pandas, NumPy, Matplotlib, Scikit-learn, Jupyter Notebook, Apache Spark, SQL
